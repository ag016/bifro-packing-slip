# Security policy

## Reporting a vulnerability

Please do not open a public issue for a suspected security vulnerability.

Email [info@bifro.ca](mailto:info@bifro.ca) with the subject **Security report: Bifro Packing Slips**. Include the affected version or commit, the steps needed to reproduce the issue, its likely impact, and any suggested remediation. Do not include real customer or personal data.

We will acknowledge a complete report as promptly as practical, normally within five business days, and will coordinate responsible disclosure when a fix is required. Please allow a reasonable remediation period before publishing details.

## Supported version

Security fixes are made against the latest version on the default branch. Because self-hosted installations can differ, operators are responsible for reviewing and applying upstream fixes to their own copy.
